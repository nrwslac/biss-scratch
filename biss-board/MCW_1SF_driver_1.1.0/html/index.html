mcw_1sf_driver Documentation

Description

This sample code proposes a driver for implementing an SPI master to communicate with iC-MCW as SPI slave.

Getting Started

  1. Put the files mcw_1sf_driver.h and mcw_1sf_driver.c into the project folder.
  2. Add the line #include "mcw_1sf_driver.h" to the application source file (e.g. main.c).
  3. Implement the function definition of all Wrapper_Functions provided by the header file mcw_1sf_driver.h.

Defining Wrapper Functions

A generic example for a definition of Wrapper_Functions is shown below, which is to be adapted with respect to the individual hardware used.

void mcw_spi_transfer(uint8_t *data_tx, uint8_t *data_rx, uint16_t datasize) {
digitalWrite(NCS_PIN, LOW);
for (uint8_t i = 0; i < datasize; i++)
data_rx[i] = SPI.transfer(data_tx[i]);
digitalWrite(NCS_PIN, HIGH);
}
